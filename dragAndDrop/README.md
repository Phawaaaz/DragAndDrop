# Todo List App with Drag and Drop

## Features

- Add new tasks
- Edit tasks
- Delete tasks
- Reorder tasks with drag and drop

## Usage

1. After cloning and installation,
2. Start your development with
   ```sh
   npm run dev
   ```
3. Open your browser and go to `http://localhost:3000`
